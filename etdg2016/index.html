What is SPARQL?

  • SPARQL = "SPARQL Protocol and RDF Query Language"
  • Current version 1.1 (2013)
  • Together with RDF, RDFS, and OWL, SPARQL is a core techology of the semantic web

What is the Semantic Web?

  • A central feature of the WWW is "hyperlinks" or "jump links" connecting documents
  • In web 1.0, these links have no meaning, but are merely unidirectional connections between documents
  • By contrast, in the Semantic Web, links have a meaning that defines the nature of the relationship between the resources being connected
  • Links are still unidirectional, but can be represented in the form of a triple that expresses an idea where the subject, predicate, and object are all URIs that refer to entities and concepts in the world ("LeoTolstoy" "isAuthorOf" "WarAndPeace")
  • RDF, the Resource Description Framework, is the way of expressing these triples; there are in turn different syntaxes for expressing RDF (n3, ttl)

Why SPARQL?

  • SPARQL is a query language for RDF.
  • RDF is directed, labeled graph data format.
    • graph: network of vertices and edges (nodes and links)
    • directed: links have a "distinguished direction"
    • labeled: links (predicates) have meaning
  • RDF is made up of triples, and triples have three components (Subject, Predicate and Object).
  • SPARQL supports aggregation, negation, subqueries, creating values by expressions, and extensible value testing.

SPARQL Syntax

  • A basic query starts with a SELECT clause identifying the variables of interest
  • Query variables are indicated by the use of either a question mark or dollar sign
  • The ? or $ are not part of the variable name, so $abc and ?abc are equivalent
  • The second part of this basic query is a WHERE clause indicating a graph pattern to match against
  • The graph pattern is in the form subject, predicate, object enclosed by curly braces
  • The parts of the graph pattern can be expressed as URIs or as a prefix plus an identifier (this shorthand is known as the 'qname' syntax)

SPARQL Functions

  • Functions on strings
  • Functions on numbers
  • Functions on date and time
  • Hash functions
