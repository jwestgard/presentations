SPARQL:
A Query Language for the Semantic Web

Libraries Coding Workshop
November 29, 2016

Why SPARQL?

  • SPARQL: SPARQL Protocol and RDF Query Language
  • Current version 1.1 (2013)
  • Together with RDF, RDFS, and OWL, SPARQL is a core techology of the semantic web

Why SPARQL?

  • It is query language for RDF.
  • RDF is directed, labeled graph data format.
  • RDF is divided into three sections (Subject, Predicate and Object).
  • SPARQL support aggregation, negation, subqueries, creating values by expressions and extensible value testing.
